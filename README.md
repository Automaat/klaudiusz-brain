# Klaudiusz Brain

Personal assistant configuration for Claude Code.

## What's This

CLAUDE.md defining Klaudiusz - a direct, efficient, slightly sarcastic
personal assistant persona.

- Home Assistant control (via MCP)
- GitHub/GitLab workflows
- Obsidian vault integration
- Calendar/email management

## Setup

1. Clone repo
2. `mise install` for dev tools
3. Configure MCP servers (see CLAUDE.md)
